# Pokerface

个人用中文命令行德州扑克小游戏。无需数据库，无需图形界面，直接在终端、Codex 或其他 AI agent 环境里运行。

## 运行

```bash
npm start
```

或：

```bash
node poker.js
```

启动后按提示设置人数、初始筹码、盲注、水下模式和 AI 难度。

## 操作

| 中文命令 | 快捷命令 | 含义 |
| --- | --- | --- |
| `弃牌` | `f` | 放弃本手 |
| `过牌` | `c` | 无人下注时让过 |
| `跟注` | `c` | 有下注时补齐到当前最高下注 |
| `下注 50` | `b 50` | 下注 50 |
| `加注 120` | `r 120` | 加注到 120 |
| `全下` | `a` | 全下 |
| `状态` | `s` | 重新显示牌桌状态 |
| `退出` | `q` | 退出游戏 |

## 测试

```bash
npm test
```

测试覆盖洗牌唯一性、牌型判定、输入解析、边池结算和一手牌自动跑通。
