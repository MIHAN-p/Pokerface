# Pokerface

Pokerface 是一个中文命令行德州扑克小游戏。它不依赖数据库、浏览器或图形界面，可以直接在终端里运行，也适合在 Codex、SSH、脚本环境或其他 AI agent 环境中游玩。

## 特性

- 支持 2-9 人牌局
- 1 名真人玩家，其余为 AI 对手
- 支持自定义初始筹码、小盲注和大盲注
- 默认支持水下模式，筹码为负时也可以继续玩
- 支持简单、普通、困难三档 AI 难度
- 包含完整德州扑克流程：翻牌前、翻牌、转牌、河牌和摊牌
- 支持边池结算、全下、弃牌、跟注、加注等常见操作

## 运行

需要本机已安装 Node.js。

```bash
npm start
```

或：

```bash
node poker.js
```

启动后按提示设置人数、初始筹码、盲注、水下模式和 AI 难度。

## 操作

| 中文命令 | 快捷命令 | 含义 |
| --- | --- | --- |
| `弃牌` | `f` | 放弃本手 |
| `过牌` | `c` | 无人下注时让过 |
| `跟注` | `c` | 有下注时补齐到当前最高下注 |
| `下注 50` | `b 50` | 下注 50 |
| `加注 120` | `r 120` | 加注到 120 |
| `全下` | `a` | 全下 |
| `状态` | `s` | 重新显示牌桌状态 |
| `退出` | `q` | 退出游戏 |

多数命令支持中文和快捷输入。下注和加注需要带金额，例如 `下注 50`、`b 50`、`加注 120` 或 `r 120`。

## 测试

